### Genres

|    **Axis**|     | **Description**                                                  |     | **Value spectrum**                                                     |
|-----------:|-----|------------------------------------------------------------------|-----|------------------------------------------------------------------------|
|      source|     | cymatics, vibration domain                                       |     | mechanic, electronic, electro-mechanic                                 |
|            |     |                                                                  |     |                                                                        |

|            |     |                                                                  |     |                                                                        |
